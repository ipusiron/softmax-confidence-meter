# Softmax Confidence Meter  
Specification (SPEC.md)

## 1. 概要
Softmax Confidence Meter は、スコア付き候補集合に softmax を適用し、
得られた確率分布の形状から「判断してよいか」を測定・可視化するツールである。

本ツールは最良候補を自動的に決定しない。
判断材料を提示する“計器（Meter）”として機能する。

## 2. 画面構成
- タブ1：softmax基礎  
- タブ2：Confidence Meter  
- タブ3：関連ツール・発展  

## 3. softmax仕様
- 温度付き softmax を使用
- 温度範囲：0.1〜5.0（デフォルト 1.0）
- **入力**: スコアは任意の値（合計が1や100である必要なし）
- **出力**: 確率分布（必ず合計が1.0＝100%になる）

## 4. 自信度（Confidence）定義
正規化エントロピーを用いる。

Confidence = (1 - H / log(N)) * 100

## 5. 判定ラベル
- 70%以上：自信あり
- 40〜70%：判断が分かれる
- 40%未満：判断困難

## 6. 非目標
- 自動決定しない
- 外部処理を実行しない
